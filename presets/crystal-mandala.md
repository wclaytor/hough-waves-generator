# Crystal Mandala

Geometric circular pattern with coarse binning optimization for clean, symmetrical lines.

## Settings

```
pattern: circle
numPoints: 120
threshold: 7
thetaStep: 0.5
opacity: 0.65
colorMode: rainbow
waveFrequency: 4
waveAmplitude: 150
spiralRotations: 6
spiralRadius: 0.4
circleRadius: 0.3
useCoarseBinning: true
thetaBinSize: 2
rhoBinSize: 5
useDynamicThreshold: false
useBlurring: false
darkMode: false
```
