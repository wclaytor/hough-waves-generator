# Crystal Mandala

Geometric circular pattern with coarse binning optimization for clean, symmetrical lines.

## Settings

```
pattern: circle
circleRadius: 0.3
numPoints: 120
threshold: 7
thetaStep: 0.5
opacity: 0.5
colorMode: rainbow
useCoarseBinning: true
thetaBinSize: 2
rhoBinSize: 5
useDynamicThreshold: false
useBlurring: false
darkMode: true
```
