# Crystal Mandala

Geometric circular pattern with coarse binning optimization for clean, symmetrical lines.

## Settings

```
pattern: circle
numPoints: 120
threshold: 25
thetaStep: 0.5
opacity: 0.12
colorMode: rainbow
useCoarseBinning: true
thetaBinSize: 2
rhoBinSize: 5
useDynamicThreshold: false
useBlurring: false
darkMode: false
```
