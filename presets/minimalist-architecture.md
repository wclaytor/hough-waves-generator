# Minimalist Architecture

Clean grid pattern with dynamic threshold for bold, architectural lines.

## Settings

```
pattern: grid
numPoints: 40
threshold: 35
thetaStep: 1
opacity: 0.90
colorMode: mono
useCoarseBinning: false
useDynamicThreshold: true
thresholdMultiplier: 0.25
useBlurring: false
darkMode: false
```
